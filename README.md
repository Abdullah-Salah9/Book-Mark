# Bookmaker App

A simple CRUD-based bookmark manager built with **HTML, CSS, Bootstrap, JavaScript, and Local Storage**.  
Easily add, edit, delete, and save bookmarks in your browser with a responsive interface.

---

## Features

- **CRUD Operations**: Create, Read, Update, Delete bookmarks.
- **Validation**: Ensures valid bookmark names and URLs.
- **Local Storage**: Saves bookmarks persistently in the browser.
- **Responsive UI**: Mobile-friendly design using Bootstrap.
- **Icons**: Clean interface using Font Awesome.

---

## Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Local Storage API
- Font Awesome
